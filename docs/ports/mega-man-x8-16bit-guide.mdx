### Install the Mega Man X8 16-bit App
- Unzip the `MegaManX8-16bit-UWP.zip` file.
- Install the `MegaManX8-16bit.msix` file via the dev portal.
- Install the `Microsoft.VCLibs.x64.14.00.appx` dependency file.

### Launch Mega Man X8 16-bit
- Once in-game, use **X** to confirm selections in the menus.
- This can be remapped in **Key Config**.

### Additional Notes
- **D-pad** movement can be remapped to the **Left Stick** in **Key Config**.
- The **CRT filter** can be applied in **Options**.
